# React + Vite Student UI

This is the student-facing frontend built using **React + Vite**.

## Features
- Vite for fast development
- React with HMR
- ESLint setup
- Easy integration with backend API
- Modular folder structure

## Available Plugins
- @vitejs/plugin-react → Babel-based Fast Refresh
- @vitejs/plugin-react-swc → SWC-based Fast Refresh

## React Compiler
The React Compiler is not enabled by default because it impacts build and dev speed.
To enable it, follow:
https://react.dev/learn/react-compiler/installation
